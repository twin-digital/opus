# Grinbox

Personal email triage. Watches a user's inbox, surfaces what matters,
summarizes the rest, and applies categorical metadata to keep the
mailbox uncluttered.

Grinbox is the next-generation successor to Mailminder (in
maintenance under `../mailminder/`).

The name: _the smile when your email doesn't overwhelm you._

## Status

In design. No application code yet — see `docs/` for the planning
artifacts.
